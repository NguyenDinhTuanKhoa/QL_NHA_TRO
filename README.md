# QL_NHA_TRO

Hệ thống quản lý nhà trọ - Tra cứu phòng trọ

## Công nghệ sử dụng
- Frontend: React.js
- Backend: Node.js, Express
- Database: MySQL

## Cài đặt

### Lần đầu tiên
```bash
cd Backend
npm install

cd ../Frontend
npm install
```

### Chạy dự án
```bash
# Terminal 1 - Backend
cd Backend
npm start

# Terminal 2 - Frontend
cd Frontend
npm start
```
